== Description ==

MapifyLite is an elite plugin for WordPress that implements fully-customized maps on your site. 

With buttery-smooth maps, enhanced graphics and pop-up galleries, MapifyLite is perfect for creating a store locator, journals, and more! Seamlessly upgrade to MapifyPro to enable routes, tours, custom maps, and faster 3D vector maps.

== Installation ==

1. Click Install, then click Activate.

2. Maps: Add your map

3. Map Locations: Add and edit locations to maps as needed. Or install MapifyPro for bulk uploading using a CSV file from Excel.
